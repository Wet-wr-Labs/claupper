# Agentic Remote

A one-handed Flipper Zero remote for Claude Code and AI coding agents. Five buttons. No keyboard required.

Built by Kasen Sansonetti & Wetware Labs.

## How It Works

Press 1 to approve, 2 to decline, 3 for alternatives, OK to confirm. Hit Down for voice dictation. Double-click Up or Down to page scroll. Double-click OK to switch terminal windows. Hold Left+Down to open Macros. The whole point is you never touch your keyboard.

## Features

- **Remote Mode** with single, double, and triple-tap actions
- **Macros** from SD card (up to 24, hold to scroll, 7 preset packs)
- **Offline Manual** covering all Claude Code features (7 categories, 29 sections)
- **Quiz Mode** with Easy/Medium/Hard difficulty
- **Settings**: OS (Mac/Win/Linux), haptics, LED, double-click speed
- **Voice dictation** via consumer HID

## Button Map

- **Left**: 1 (approve) / double: clear line / triple: end of line / hold: backspace
- **Up**: 2 (decline) / double: page up / triple: up arrow
- **Right**: 3 (other) / double: prev command / triple: right arrow
- **OK**: Enter / double: switch window
- **Down**: Voice dictation / double: page down / triple: down arrow
- **Back**: Return home / hold: Escape

## Combos

- **Right + Down** (hold both): Show hotkey overlay
- **Left + Down** (hold both): Open Macros menu
